# 学生小组管理系统

## Core Features

- 学生信息增删改查

- 小组创建和管理

- 学生小组关联管理

- 图形化数据展示

## Tech Stack

{
  "Desktop": "Avalonia UI (C#/.NET) with MVVM architecture"
}

## Design

现代化桌面应用设计，采用蓝色主题的卡片式布局，包含侧边导航菜单、数据表格展示和滑出式编辑面板，支持拖拽操作进行学生小组关联管理
